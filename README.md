# MokshaGPT

Welcome to the MokshaGPT repository. This project contains both the backend and graphical user interface (GUI) components.

## Project Structure

- `backend/`: The main backend services and logic.
- `gui/`: The frontend GUI implementation.
- `docs/`: Design documents, guides, and historical development logs.
- `terraform/`: Infrastructure as Code (IaC) files.
- `sample_data/`: Sample datasets for testing.
- `sample_trades/`: Sample trade logs.
- `evaluation/`: Scripts and models for evaluating the system.
- `jupyter-notebook/`: Jupyter notebooks for data analysis or experiments.

## Getting Started

Please refer to the documentation within the `backend/` and `gui/` directories for specific setup and execution instructions.
